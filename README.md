# 🧮 Flutter Calculator App

A simple and efficient calculator app built using Flutter. It supports basic arithmetic operations and uses a custom infix-to-postfix conversion algorithm for accurate expression evaluation.

---

## 🚀 Features

- Basic operations: +, −, ×, ÷, %
- Infix to postfix expression conversion
- Stack-based evaluation logic
- Smart result formatting (removes unnecessary decimals)
- Clean and responsive UI
- Real-time input display

---

## 🛠️ Tech Stack

- Flutter (Dart)
- Stack data structure
- Custom expression parsing algorithm
